## Pre-requisites
1. MongoDB CLI.
2. Node v18.x
3. Define the ENV file in the root project. A sample is attached for reference (.env.example)
4. MongoDB compass - in case you want to view the DB data..

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

#build 
$ npm run build 

# production
$ npm run start
```
